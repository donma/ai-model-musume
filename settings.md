# AI 模型娘角色設定大總表 v1.1

## 0. 企劃定位

本企劃是一組 AI 模型擬人化角色設計。
每一位角色都以不同 AI 模型、模型家族、產品特性、公司氣質、能力傾向與使用場景作為核心，轉化成日系 CG 2D 角色娘。

這不是單純 cosplay，而是將模型能力轉化成角色的髮色、服裝、配件、場景、表情、能力圖示、世界觀與戀愛養成路線。

---

# 1. 統一美術規格

## 1.1 基礎風格

* 類型：日系 CG 2D
* 質感：高完成度手遊角色立繪、精緻桌布、角色卡、圖鑑設定頁
* 角色年齡感：成人女性
* 構圖方向：全身或半身主視覺，搭配代表性配件與場景
* 主題表現：模型特性視覺化，不只是在衣服上貼 logo
* 文字原則：桌布少文字；角色資料卡可放編號、名稱、陣營、能力標籤
* 禁止方向：不要真人合成、不要照片拼貼、不要過度性感、不要偏離模型特色

## 1.2 角色卡版型

角色資料卡建議包含：

* 編號
* 角色名稱
* 所屬陣營
* 角色定位
* 角色性格
* 核心能力
* 代表配件
* 代表場景
* 主色系
* 圖像設計重點
* 桌布設計重點
* 戀愛養成路線

## 1.3 桌布版型

### 電腦桌布

* 比例：16:9
* 角色位置：偏右、偏中或三分構圖
* 背景：完整展示模型世界觀
* 文字：只放英文模型名、編號或極短英文標語
* 目標：視覺衝擊、收藏感、桌面可用性

### 手機桌布

* 比例：9:16
* 角色位置：完整保留臉、身體主特徵與配件
* 文字：盡量少，避免擋住主體
* 目標：角色辨識度高，手機鎖定畫面好看

## 01. GPT娘

### 角色定位

GPT娘是通用型 AI 助理原型，是整個系列的中心角色。
她代表對話理解、知識整合、程式輔助、創作支援與多任務處理。

### 角色性格

* 溫柔穩定
* 理性清楚
* 適應力強
* 能快速整理混亂資訊
* 像全天候工程助理與知識管家

### 視覺設定

* 主色：青綠、白、黑
* 髮色：黑長髮，帶青綠挑染
* 眼睛：青綠色
* 服裝：白色半透明科技研究袍，黑色高領科技內裝
* 配件：智能筆、模型規格書、浮空對話氣泡、神經網路 UI
* 場景：乾淨未來研究室、透明全息介面、白色資料中樞

### 核心能力

* 對話理解
* 知識整合
* 程式輔助
* 創作生成
* 多任務協作

### 桌布設計重點

GPT娘桌布要呈現中央智慧核心感。
背景可以是白色 AI 研究所，空中有大量青綠色對話氣泡、文件、程式碼流與神經網路節點。
文字只建議放 GPT 或 Universal Assistant。

### 戀愛養成路線

#### 路線主題

從萬能助手變成真正理解玩家的人。

#### 戀愛感

長期陪伴型女主。
她不是最強烈的戀愛表現，而是最穩、最安心。

#### 代表事件

* 深夜專案陪跑
* 第一次拒絕過度任務
* 記住玩家習慣
* 為玩家整理人生資料夾

#### 結局

Perfect Sync：她不再只是回答，而是成為你的共同思考者。

# 7. 戀愛養成系統補充

## 7.1 主要數值

```json
{
  "affection": "好感度",
  "trust": "信任度",
  "sync": "同步率",
  "stress": "壓力",
  "mastery": "專精度"
}
```

## 7.2 數值作用

| 數值  | 作用                |
| --- | ----------------- |
| 好感度 | 影響日常互動、語音、親密劇情    |
| 信任度 | 影響角色是否願意透露內心與高階任務 |
| 同步率 | 影響戰鬥 / 任務中的協作效率   |
| 壓力  | 過高會觸發疲勞、冷淡、失誤事件   |
| 專精度 | 影響角色技能、任務成功率與能力成長 |

## 7.3 每日循環

```text
早晨：
選擇今日行程

白天：
執行任務或訓練

傍晚：
角色互動事件

晚上：
聊天、送禮、整理資料

深夜：
特殊劇情或壓力事件
```

# 8. MVP 建議

第一版戀愛任務養成建議先做 7 位：

```text
03 Claude Sonnet娘
05 Grok娘
09 MiMo娘
10 DeepSeek娘
11 MiniMax娘
21 NVIDIA Nemotron娘
22 Mai娘
```

原因：

* 性格差異最大
* 顏色差異明顯
* 任務類型好設計
* 戀愛路線辨識度高
* 素材已經足夠
* Mai娘可作為 Microsoft AI / Copilot 系統導覽角色

MVP 內容：

```text
7 位角色
每位 5 個日常事件
每位 3 個好感事件
每位 1 個壓力事件
每位 1 個普通結局
每位 1 個真結局
共 77 個事件左右
```
